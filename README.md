# 데이터베이스언어(Database Language)
- 정보시스템의 근간인 데이터베이스 시스템의 개념을 살펴보고, 데이터베이스를 사용하는 방법에 대해 배운다. 특히, Oracle 데이터베이스 시스템을 사용하여 SQL과 PL/SQL 언어를 중점적으로 학습한다.
